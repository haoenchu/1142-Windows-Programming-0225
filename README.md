# BMI計算機
--------------
視窗程式設計(II) 上課範例

可以輸入身高(cm)、體重(kg)，按下計算按鈕後，會顯示BMI值、BMI分類。

功能說明:
1.textbox會提前預覽，導覽使用者各個輸入項目

2.放置BMI圖片供使用者參考

3.輸出計算結果與特定的警語
